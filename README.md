SPACE SHOOTER GAME

Description:
Space Shooter is a game in which the user has to control his player using A/D or left right arrow keys. The enemies are spawning randomly. 
User has to save his player and attack the enemies using bullets by pressing space bar. When the player successfully attacks the enemy the score gets increased but when the player misses the enemy the
score gets decreased. And when the enemy hits the player the game gets over.

Features:
Colorful Interface
Sound Effects
Player Customization
Multiple Levels
Different Difficulty Levels

Technologies Used:
C Language
